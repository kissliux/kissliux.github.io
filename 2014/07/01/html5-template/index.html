简介

前端开发模板，自动化任务管理。html5 development template, auto task manager

本项目为笔戈科技开发团队前端开发模板。方便快速进行项目开发，打包发布。

使用方式

前端技术

  • html5
  • bootstrap 3 模板
  • jquery
  • lazyload
  • less
  • grunt
  • bower

grunt任务：

  • clean 清理文档格式
  • less 编译成css
  • cssmin css压缩
  • uglify js文件压缩
  • compress 压缩成 .zip文件
  • copy 把所有资源文件拷贝到 build目录，方便项目发布到CDN
